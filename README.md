# First-Stage-Gradstack - Aurelia Sari
Hi! This repo contains code for Gradstack first stage take home project.
It is a simple frontend profile card and member directory view of the Gradstack Community Hub.

## How to Run Locally
### 1. Clone the repository locally
git clone the-repo-url

### 2. Open index.html
Open index.html in your preferred code editor, or open it from your file explorer to view it locally.
